Go-Mix Programming Language

A high-performance, interpreted language implemented in Go. Features dynamic typing with static safety, lexical closures, first-class functions, and a comprehensive standard library with 100+ built-in functions.

Why Go-Mix?

Designed for rapid prototyping, education, and embedding as a scripting layer in Go applications.

High Performance

Implemented in Go with efficient Pratt parser and optimized evaluation engine for fast execution.

Hybrid Type System

Three variable kinds: var (dynamic), let (static), and const (immutable) for flexibility and safety.

First-Class Functions

Full support for higher-order functions, closures, and functional programming patterns.

Rich Collections

Arrays, lists, tuples, maps, and sets with comprehensive manipulation functions and functional operations.

Comprehensive Standard Library

100+ built-in functions across 17 packages: strings, math, file I/O, HTTP, regex, JSON, and more.

Interactive REPL

Built-in Read-Eval-Print Loop for rapid prototyping and experimentation with line-by-line execution.

Clean, Expressive Syntax

Familiar syntax combining the best of Go, JavaScript, and Python with powerful features.

fibonacci.gm
{% highlight go %} // Recursive Fibonacci with memoization fn fibonacci(n) { if (n <= 1) { return n; } return fibonacci(n - 1) + fibonacci(n - 2); } // Higher-order function fn makeMultiplier(factor) { return fn(x) { return x * factor; }; } var triple = makeMultiplier(3); // Main execution for (var i = 0; i < 10; i = i + 1) { printf("fib(%d) = %d, triple = %d\n", i, fibonacci(i), triple(i)); } {% endhighlight %}
